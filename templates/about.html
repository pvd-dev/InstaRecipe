About

InstaRecipe is a simple hobby project by me, Peter. By combining my love for coding and food, I set out to better understand the Flask and Bootstrap frameworks and integrate to an API. I know, technical mumbo jumbo, the gist of it is: This was a rather complex coding project for me and I learned a lot. The recipes come from the breakfastAPI see Github if you want to know more. If you came here through a Facebook advertisement, that is because I'm trying to work out if responsive design (where the screensize scales to suit your device) actually leads to more engagement. And so I threw $10,- at facebook adds to generate some clicks

Breakfast API on Github
Techstack

I use Flask in Python to call on the Breakfast API, which returns a recipe with the ingredients and it's cooking time. I use the Rapid API to return a picture based on the Recipe's name (Though I only get 100 free calls per day). Then I used bootstrap (for the first time 🎉) to render a website. The preview for sharing on socials was done using OpenGraph.
